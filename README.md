# 寻乌调查研究导师 🔍

> 基于毛泽东1930年《寻乌调查》方法论，让AI成为你的调研导师。

---

## 这是什么？

一个 [EasyClaw](https://easyclaw.com) skill，让AI像导师一样，用毛泽东《寻乌调查》的方法论，一步步指导你完成深度调研。

不是让你背理论，是真的帮你做调查、设计访谈、分析数据、写出报告。

---

## 核心能力

- 🎯 **问题澄清**：追问你的真实调研动机，找到你的"门外汉"区域
- 📐 **结构设计**：三层递进调查法（马上观花→下马观花→解剖麻雀）
- 🎤 **访谈指导**：定制访谈提纲，分析访谈结果，追问深层问题
- 📊 **数据督导**：督促量化，指出"走马看花"式调研的漏洞
- 📝 **报告写作**：参照寻乌五章结构，指导写出有说服力的报告

---

## 背后的方法论

《寻乌调查》是毛泽东1930年5月在江西寻乌县做的大规模社会调查，8万字，被认为是中国共产党调查研究的典范。其方法论包括：

1. **正确理论指导 + 实事求是**：带着假设去，允许事实推翻假设
2. **一般性调查 + 典型性调查结合**：宁可把一个地方研究透，也不浅尝很多
3. **调查主题明确，有的放矢**：所有问题服务于一个核心问题
4. **在调查中发现和解决问题**：不只是收集，而是思考
5. **甘当小学生**：放下姿态，真正把被调查者当老师

## 如何使用

### 在 EasyClaw 中安装

```bash
# 克隆本仓库
git clone https://github.com/your-username/xunwu-research-mentor

# 注册 skill
python ~/.easyclaw/extensions/create-skill/scripts/easyclaw_register_skill.py ./xunwu-research-mentor
```

### 触发方式

在 EasyClaw 对话中，说：
- "我想做一个调研……"
- "帮我设计一份访谈提纲"
- "我的调研报告怎么写"

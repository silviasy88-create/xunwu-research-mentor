# 如何保证调研质量

> 信度、效度、饱和点、三角验证、偏差识别——让你的调研经得起追问

---

## 核心问题：你的调研可信吗？

调研结束后，必须能回答这四个问题：
1. **你测量的是你想测量的吗？**（效度）
2. **换一个人来调查，结论会一样吗？**（信度）
3. **你访谈的人代表了整体吗？**（代表性）
4. **你的结论能推广到其他地方吗？**（外部效度）

---

## 一、效度（Validity）：测到了真实的东西吗？

**内容效度**：你的问题覆盖了你想研究的全部维度吗？
- 检查方法：把你的访谈提纲列出来，对照你的核心问题，逐条问"这道题回答了核心问题的哪个部分"
- 寻乌的做法：毛泽东围绕"商业政策"和"富农问题"这两个核心，设计了覆盖交通、市场、阶级、土地的完整问题体系

**面孔效度**：专业人士看到你的调研设计，觉得合理吗？
- 检查方法：把调研设计给一个懂这个领域的人看，问"有没有明显的遗漏"

**构念效度**：你用来测量的"代理变量"真的代表了你想测量的概念吗？
- 经济学常见陷阱：用"收入"代表"生活质量"——但有人高收入高支出
- 寻乌的处理：不只测收入，还测资产、负债、雇工关系、土地占有——多维度组合

---

## 二、信度（Reliability）：结果稳定可重复吗？

**访谈者信度**：你的提问方式会不会影响受访者的回答？
- 常见问题：引导性提问（"你是不是觉得……"）
- 处理方法：尽量用开放性问题，"你怎么看""能说说这件事的经过吗"

**跨受访者一致性**：同一问题问不同人，口径能对上吗？
- 寻乌的做法：毛泽东把同一事实（如某年的牛市规模）跟多个参与者核实
- 现代操作：同一个核心事实，至少用两个独立信息来源印证

**记录信度**：你的访谈记录是否准确？
- 最好：录音+逐字转录
- 次好：访谈后30分钟内写详细笔记
- 差：依赖记忆事后重建

---

## 三、三角验证（Triangulation）：多角度印证

**数据三角**：同一事实，用三种不同来源印证
```
寻乌牛市年交易量
├── 税收承商说：每年标约4500头以上
├── 毛泽东现场计算：3320头（保守估计）
└── 牛税记录：政府每年收1740元
→ 三个来源相互印证，结论可信
```

**方法三角**：同一问题，用不同调查方法
- 访谈（主观陈述）
- 观察（现场行为）
- 文件分析（历史记录、账目）
- 寻乌示范：毛泽东开调查会（访谈）+ 实地走访各店铺（观察）+ 核查税收数据（文件）

**时间三角**：不同时间点调查同一对象，验证稳定性
- 毛泽东在调查结束一年后（1931年）整理材料，写下自己发现的不足

## 四、偏差识别与处理

### 选择性偏差（Selection Bias）

**你访谈到的人，真的代表了你想了解的群体吗？**

常见陷阱：
- **生存者偏差**：只访谈还在做的人，没有问到失败的人
  - 例：研究创业，只问成功的企业家，没问倒闭的
  - 寻乌对应：毛泽东既问了生意好的郭友梅，也关注了已经衰落的市场
- **可及性偏差**：只访谈容易接触的人（领导、愿意接受访谈的人）
  - 处理：主动寻找不愿意说话的人，追问原因
- **信息人偏差**：你的关键信息人有自己的立场，会引导你的判断
  - 寻乌处理：古柏带路，但毛泽东主动找了不同立场（商人、贫农、干部）的人

### 社会期望偏差（Social Desirability Bias）

**受访者告诉你他们认为你想听的，而不是真实情况**

识别信号：
- 所有受访者都给出正面回答
- 受访者在问到敏感话题时停顿、改变语气
- 受访者的陈述和你观察到的现象不符

处理方法：
- 问间接问题（"其他人一般怎么做？"代替"你怎么做？"）
- 建立足够的信任关系再问敏感问题
- 用行为数据验证口头陈述

### 霍桑效应（Hawthorne Effect）

**受访者因为知道被研究，改变了行为**

寻乌的处理：毛泽东以红军指挥官身份出现，受访者可能有顾虑——所以他特意营造轻松气氛，让大家"像拉家常一样"

现代处理：
- 长期田野（让对方习惯你的存在）
- 非正式观察（不告知被观察）
- 与正式访谈结果互相比对

## 五、饱和点：什么时候可以停？

**理论饱和（Theoretical Saturation）**：当新的访谈不再产生新的概念或维度时，可以停止

判断方法：
1. 把每次访谈的新发现记录下来
2. 当连续3-4次访谈没有出现新的主题，说明接近饱和
3. 对新类型的受访者做1-2次验证访谈，确认无新信息

一般参考数量：
| 调研类型 | 建议访谈量 |
|---------|-----------|
| 探索性调研 | 同类受访者5-8人 |
| 中等深度调研 | 同类受访者10-15人 |
| 深度质性研究 | 15-25人（跨多类群体） |
| 单案例解剖 | 1个案例，反复访谈 |

---

## 六、成员检验（Member Checking）

**把你的初步结论给受访者看，问他们"我理解对了吗？"**

操作步骤：
1. 访谈结束2-3周后，整理出你的初步发现
2. 选择2-3位关键信息人，把发现分享给他们
3. 问："这是你想表达的意思吗？有没有我误解的地方？"
4. 根据反馈修正，记录分歧（不一定全改）

注意：成员检验可能遇到阻力——受访者可能不想被记录成"这样的人"，这本身也是信息。

---

## 七、快速自检清单

调研完成后，对照这个清单：

```
[ ] 我的核心问题是什么？（一句话写出来）
[ ] 我的受访者代表了哪些立场？（有没有明显缺失的群体）
[ ] 每个关键事实，我有没有至少两个独立来源印证？
[ ] 我的访谈记录是完整的，而不是依赖记忆的？
[ ] 我发现了哪些颠覆我原有预设的信息？
[ ] 我有没有做成员检验？
[ ] 我的结论能推广到哪里？不能推广到哪里？
[ ] 我有没有诚实地说明调研的局限？
```
